Augmio

Augmio is a new Augmented Reality media platform. Augmio uses videos, time synced with AR content to provide the user a new interactive way to consume media.

As one of the only two primary developers working on this app I have been able to work on multiple parts, and dive in to new languages I never had experience with.

My primary focus has been on the Android version of the app. We utilize "Unity as a Library", which is compiling a Unity project into a library to be inserted into a native application. This allows us to use Unity's AR Foundation, a cross platform AR framework, to interact with object in realtime, while also taking full advantage of the Android and iOS native UI to build a seamless flow throughout the app.

My secondary focus is on our backend infrastructure. Using a MySQL database in conjunction with a NodeJS web-server and RESTful API to upload and maintain the content and users on our platform. I spent time linking our web-server with our Amazon S3 bucket for media storage, and to Auth0, which allows users to login using a social account, and keeps their data secure.
