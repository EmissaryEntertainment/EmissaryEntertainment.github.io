Wormhole

I started my time working on Wormhole as a Junior Unity Developer, working with a team of seven total developers, consisting of two other Unity developers, two Android developers, and two iOS developers. After roughly six months, the team had shrunk to 3 people and I had been promoted to the Unity Tech Lead, which allowed me to work more closely with the Project Manager and other higher ranking members of the Wormhole team.

I spent a majority of my first few months focused on the UI experience within the app. Making sure the flow of the app was user friendly, and robust. As my time on the project progressed I was given more daunting tasks such as working with GPS data to properly place users and other objects onto the globe and map that were available to the users. 

Eventually focus was turned toward using Unity's AR Foundation framework to build a robust avatar system so that users could see each other in the real world, in real time. This led to much time being spent in R&D for the best solution for us. We began using Google Cloud Anchors to position users relative to a localized object.
