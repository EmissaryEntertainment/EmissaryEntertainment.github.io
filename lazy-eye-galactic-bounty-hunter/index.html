Lazy Eye Galactic Bounty Hunter

App Store Description:

This is an "endless runner" style video game, set in space, where the player avoids blaster bolts and destroys boss monsters in a set of different, increasingly difficult boards, while also serving as a vision therapy tool for treatment of amblyopia (lazy eye).

I was the primary gameplay developer for this project. As so, it was my job to ensure the main game loop was a fun, interactive experience for children and adults alike, which helps this game feel less like a medical tool, and more like what it is, a game. This game is built using the Unity Engine, and C# programming language, in conjunction with a tool called Zenject which allowed us to utilize the Dependency Injection design pattern to minimize the work needed for objects to interact with each other in a seamless manner.
