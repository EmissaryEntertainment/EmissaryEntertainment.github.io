Drop Squad: The Escape!

App Store Description:

Drop Squad™ is the most challenging action & puzzle game you’ve NEVER played. It takes brain power and ball mastery to outsmart, outbuild, and outscore friends and foes.

I joined this project about 3/4 through it's development, and experiencing a fast onboarding process to an active project was a great learning experience. My initial primary task was to create a playable ad for use when advertising. It was left to my discretion to find a suitable engine to build and deploy the ad for use. After testing many candidates I landed on Cocos Creator. This tool was able to mimic a lot of what the main game engine (Unity) had to offer, while allowing a fast single page html build of the app for deployment. Through the html build pipeline in Cocos, and a custom minification script written in Python, I was able to develop and deploy a playable ad into a single page html file less than 2mb in size to fit within Facebook ad restrictions.

After the completion of the playable ad, I was asked to continue working on the main project, being built inside the Unity Engine, and work with the clients custom backend solution Elements. This helped me further my knowledge within Unity, customizing to fit their game structure and flow, as well as learn about scalable backend architecture using long running coroutines and a docker distributed backend.
