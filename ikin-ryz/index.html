IKIN Ryz

This project was designed to take the technology seen in the video below, and put it into the hands of the consumer by shrinking it down to work on a phone, and couple with Unity to allow for anyone to create content for the platform on their own.

As a primary developer for the RYZ device SDK, I have been working closely with the company and their hardware team to create a seamless experience for users to create content for this platform. My primary work has been done using Python, C++ and the Unity Engine with C#. Building this project has allowed me to thrive and learn in ways I could never have hoped to this early in my career.

Using Python I was able to create a simulation of the hardware, using a TCP connection to talk between the primary SDK and the simulator with a network protocol of messages that allow us to show what can be done with the technology. This has been vital during development of the final product. 

Learning and utilizing other technologies such as CMake, Jenkins, and Doxygen have allowed use to streamline the process for developers across many platforms. (Mac, Linux, Windows, Android, and iOS)
